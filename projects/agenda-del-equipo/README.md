# Agenda del equipo

Web part SPFx para mostrar reuniones, demos e hitos próximos del equipo en formato compacto y filtrable.

## Capacidades

- soporte para `StaticConfig`, `Calendar`, `SharePointList` y `JsonUrl` same-origin
- agrupación temporal por `Hoy`, `Mañana`, `Próximos días` y `Pasados`
- filtro por tipo de evento desde la propia UI
- estados `loading`, `ready`, `empty`, `partialData` y `error`
- `WebPartErrorBoundary` en el montaje raíz

## Validación prevista

- `npm run build`
