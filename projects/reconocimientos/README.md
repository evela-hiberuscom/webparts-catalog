# Reconocimientos

Web part SPFx para mostrar un feed ligero de agradecimientos y logros recientes del equipo.

## Capacidades

- soporte para `StaticConfig`, `SharePointList` y `JsonUrl` same-origin
- estados `loading`, `ready`, `empty`, `partialData` y `error`
- orden descendente por fecha con degradación segura cuando faltan foto, mensaje o enlace
- `WebPartErrorBoundary` en el montaje raíz

## Validación prevista

- `npm run build`
