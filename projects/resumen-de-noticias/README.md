# Resumen de noticias

Web part SPFx para destacar noticias publicadas en `Site Pages` con lectura compacta y prioridad visual para la pieza más reciente.

## Capacidades

- lectura de noticias promovidas desde una biblioteca de páginas de SharePoint
- orden editorial por fecha con destacado opcional para la primera pieza
- degradación segura cuando faltan extracto, imagen o enlace
- estados `loading`, `ready`, `empty`, `partialData` y `error`
- `WebPartErrorBoundary` en el montaje raíz

## Validación prevista

- `npm run build`
