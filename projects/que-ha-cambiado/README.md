# que-ha-cambiado

Web part SPFx para mostrar cambios recientes desde una lista o biblioteca de SharePoint.

## Capacidades

- Lee elementos recientes desde `SharePoint List` o `SharePoint Library`.
- Normaliza título, tipo, fecha, resumen y URL de apertura.
- Filtra por tipo por defecto.
- Muestra estados `loading`, `empty`, `partialData` y `error`.

## Configuración

- `title`
- `description`
- `sourceKind`
- `listTitleOrUrl`
- `defaultTypeFilter`
- `maxItems`

## Validación

- `npm run build`
