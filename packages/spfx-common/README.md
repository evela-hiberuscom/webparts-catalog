# @paquete/spfx-common

Paquete interno compartido para los proyectos SPFx generados desde Airtable.

Incluye:
- tokens de diseño y tema compartido
- utilidades de accesibilidad y enlaces seguros
- componentes base reutilizables
- helpers de testing
- estrategia de self-host de tipografías

Está diseñado para ser consumido desde proyectos SPFx mediante dependencia local `file:../../packages/spfx-common`.
