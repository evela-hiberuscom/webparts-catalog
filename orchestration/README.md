# Orchestration

Artefactos de coordinación para la ejecución multiagente de proyectos SPFx derivados de Airtable.

Contenido:
- `project-work-items.json`: catálogo normalizado de rows y su planificación operativa
- `issue-registry.json`: memoria central de errores, bloqueos y mitigaciones
- `decision-log.json`: decisiones persistidas para evitar ambigüedad entre workers
- `archetype-notes.md`: patrones reutilizables y pitfalls por arquetipo
- `templates/`: plantillas de documentación interna por proyecto
